Dicionário de Elementos HTML

  1. !DOCTYPE html: Define a versão do documento HTML.
  2. html: Elemento raiz que engloba todo o conteúdo da página.
  3. header: Contém metadados da página, como metatags, links para estilos CSS, informações do autor e muito mais.
    • meta charset="UTF-8": Define o conjunto de caracteres usado no documento como UTF-8, que inclui uma ampla variedade de caracteres para diferentes idiomas.
    • meta http-equiv="X-UA-Compatible" content="IE=edge": Especifica que a página deve ser exibida na versão mais recente do Internet Explorer.
    • meta name="viewport" content="width=device-width, initial-scale=1.0": Define a configuração da viewport para dispositivos móveis, garantindo um layout responsivo.
    • link rel="stylesheet" href="style.css": Importa um arquivo externo de estilo CSS chamado "style.css" para estilizar a página.
    • title: Define o título da página exibido na aba do navegador.
  4. body: Contém todo o conteúdo visível da página, incluindo texto, imagens, links e elementos estruturais.
  5. div class="container": Um contêiner que envolve todo o conteúdo visível da página para aplicar estilos CSS.
  6. header: A seção de cabeçalho da página que normalmente contém o logotipo, o título da página e a navegação.
    • h1: Título principal da página.
    • nav: Elemento de navegação que contém links para diferentes seções da página.
    • a: Links para navegar para outras páginas ou âncoras internas na mesma página.
  7. aside: Uma seção lateral que normalmente contém informações relacionadas ao conteúdo principal.
    • img: imagem
    • h2: Um subtítulo.
    • address: Um elemento HTML que fornece informações de contato.
  8. main: A seção principal do conteúdo da página.
    • section: Uma seção dentro do main que contém informações agrupadas.
    • h3: Títulos de seção.
    • p: Parágrafos de texto.
    • span: Um elemento inline usado para estilizar parte do texto.
    • ul: Lista não ordenada.
    • li: Itens de lista.
    • br: Quebra de linha.
  9. footer: O rodapé da página que geralmente contém informações de direitos autorais ou créditos.
